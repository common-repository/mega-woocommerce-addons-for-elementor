Mega WooCommerce Addons for Elementor provides awesome custom Woocommerce widgets for the Elementor page builder. Build a WooCommerce website using Elementor within minutes. Display unique WooCommerce Product layout With Mega WooCommerce Addons for Elementor.

== Description ==

Mega WooCommerce Addons for Elementor is a plugin for the popular Elementor page builder. It gives the user eleven extra WooCommerce 'elements' (widgets) to use.

[Check the Demos](https://atlantisthemes.com/elementor-addons-woocommerce/)

### [Elementor](https://wordpress.org/plugins/elementor/) page builder is required for this plugin.
